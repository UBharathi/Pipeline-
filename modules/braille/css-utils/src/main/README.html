Transformers (org.daisy.pipeline.braille.common.TransformProvider)

  • (input:css)(output:css)

    Recognized features:

    • id: If present it must be the only feature. Will match a transformer with a unique ID.
    • locale: If present the value will be used instead of any xml:lang attributes.

    Other features are used for finding sub-transformers of type BrailleTranslator that match (input:text-css).
