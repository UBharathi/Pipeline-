Transformers (org.daisy.pipeline.braille.common.TransformProvider)

  • (hyphenator:hyphen)

    Recognized features:

    • id: If present it must be the only feature. Matches a hyphenator with a unique ID.
    • hyphenator: Will only match if the value is hyphen, or if it’s a hyphenator’s ID.
    • table or libhyphen-table: A Hyphen table is a URI that can be either a file name, a file path relative to a registered table path, an absolute file URI, or a fully qualified table identifier. This feature is not compatible with other features except hyphenator.
    • locale: Matches only hyphenators with that locale.
